Keyboard Accessibility Aspect of Website (Documentation)

This website uses keyboard navigation to ensure anyone is able to use this website :D.

The aspect of keyboard support I am especially trying to achieve is Heuristic #7: Flexibility and Efficiency of Use. This keyboard design supports accessibility by allowing access and navigation methods, which is especailly important to consider when designing any website. This provides access that caters to the users who are unable to use a mouse. Using the Tab and Arrow Keys to navigate this website provides a great alternative for all audiences!

  • Use the TAB key to move between links and buttons
  • Use the UP and DOWN arrow keys to move through reading links (On the THINGS TO READ section) and scrolling up and down (on the HOME page)
  • Press ENTER to open the selected link!
